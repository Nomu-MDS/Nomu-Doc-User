# Gérer son Compte

## Accéder aux paramètres

Depuis l'onglet **Profil** en bas de l'écran, vous avez accès à toutes les options de gestion de votre compte.

---

## Modifier son profil

Appuyez sur **Modifier le profil** (Edit Profile) pour mettre à jour :

| Champ | Utilité |
|-------|---------|
| **Photo de profil** | Première impression — gardez-la à jour |
| **Biographie** | Décrivez qui vous êtes et ce que vous aimez partager |
| **Ville / Pays** | Mettez à jour si vous déménagez ou voyagez longtemps |
| **Prénom, nom, âge** | Informations de base affichées sur votre profil |

---

## Mettre à jour ses intérêts

Vos passions évoluent ? Vos intérêts Nomu aussi.

Depuis votre profil, accédez à **Centres d'intérêt** pour ajouter de nouveaux intérêts ou retirer ceux qui ne vous correspondent plus.

:::info
Mettre à jour vos intérêts améliore immédiatement la pertinence des profils qui vous sont suggérés et vous rend plus visible dans les recherches correspondant à vos nouvelles passions.
:::

---

## Visibilité du profil

Vous pouvez choisir si votre profil apparaît dans les résultats de recherche :

- **Profil visible** — Vous apparaissez dans les recherches des autres membres
- **Profil masqué** — Vous restez invisible mais pouvez toujours parcourir les profils et envoyer des messages

Cette option est utile si vous souhaitez prendre une pause ou n'êtes pas disponible pour le moment.

---

## Se déconnecter

Pour protéger votre compte, notamment sur un appareil partagé, utilisez l'option **Se déconnecter** disponible dans l'onglet Profil.

:::warning
Sur mobile, la déconnexion supprime votre session locale. Vous devrez vous reconnecter avec vos identifiants ou via Google lors de votre prochaine visite.
:::
