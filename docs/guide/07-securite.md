# Sécurité et Signalement

## Notre engagement

Nomu s'engage à maintenir une communauté **saine, bienveillante et basée sur la confiance**. Pour cela, nous comptons sur l'ensemble de nos membres pour signaler les comportements inappropriés.

---

## Signaler un utilisateur

Si vous rencontrez un comportement inapproprié — que ce soit sur un profil ou lors d'un échange par message — vous pouvez et devez utiliser la fonction **Signaler** (Report).

### Comment signaler ?

1. Accédez au **profil de l'utilisateur** concerné
2. Faites défiler jusqu'en bas du profil et appuyez sur **Signaler ce profil** (lien discret sous le bouton Contacter)
3. Un formulaire s'ouvre — saisissez la **raison** du signalement (champ obligatoire)
4. Ajoutez des **détails optionnels** pour préciser le contexte si nécessaire
5. Appuyez sur **Signaler** — votre demande est transmise immédiatement à l'équipe Nomu

:::info
Votre signalement est **confidentiel**. L'utilisateur signalé ne saura pas que c'est vous qui avez effectué le signalement.
:::

---

## Traitement des signalements

L'équipe d'administration de Nomu examine **chaque signalement avec soin** :

| Étape | Description |
|-------|-------------|
| **Réception** | Le signalement est enregistré et mis en file de traitement |
| **Examen** | Un modérateur Nomu examine le profil ou les messages concernés |
| **Action** | Des mesures correctives sont prises selon la gravité |

### Mesures possibles

- **Avertissement** envoyé à l'utilisateur
- **Restriction temporaire** des fonctionnalités
- **Blocage définitif** du compte

---

## Bonnes pratiques de sécurité

:::warning
Comme sur toute plateforme de mise en relation, faites preuve de bon sens :

- Ne partagez pas d'informations personnelles sensibles (adresse exacte, coordonnées bancaires) dans les messages
- Pour un premier rendez-vous, privilégiez un **lieu public**
- Informez un proche de votre rencontre
:::
